# Biddify | Rob Rutjes | Fontys ICT | Semester 3

## Omschrijving Project
Biddify is een applicatie die voor gebruikers makkelijk gaat maken om producten te kopen en verkopen.

Gebruikers kunnen zichzelf makkelijk registreren en inloggen. Hierna kunnen zij fotoâ€™s van hun product maken een titel, omschrijving & prijs toevoegen en het product op Biddify plaatsen. Andere gebruikers kunnen door de producten scrollen en deze kopen voor de vraagprijs. Er komt een overzicht waar gebruikers hun gekochte en verkochte producten kunnen zien.

In fase 2 van de applicatie kan er support voor live biedingen via opbod worden toegevoegd en betalingen via een externe betaling service als bijvoorbeeld Stripe.
